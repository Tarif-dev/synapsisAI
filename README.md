# Synapsis AI

AI-powered UI component generator.

## Features

- Prompt → React component generation
- Groq LLM integration
- UI schema architecture
- Deterministic code generation
- Live preview (WIP)

## Tech Stack

Frontend:
- Next.js
- Tailwind
- Sandpack

Backend:
- FastAPI
- Groq LLM
- Python

## Architecture

Prompt → LLM → UI Schema → Code Generator → React Component
